itsdangerous库加密

发表于 2021-01-13 更新于 2021-05-17 分类于 Python

itsdangerous是Python的一个库,可以实现数据的加密功能

下载

pip3 install itsdangerous

导入

from itsdangerous import TimedJSONWebSignatureSerializer

使用

# 创建加密对象,一个个参数为秘钥,第二个为加密有效时间
serializer = TimedJSONWebSignatureSerializer('secretkey',3600)

# 需要加密的内容
info = {'confirm':1}

# 加密
res = serializer.dumps(info)

# 变成utf-8编码
res = res.decode()

# 解密
reinfo = serializer.loads(res)
