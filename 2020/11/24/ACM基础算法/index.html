发表于 2020-11-24 更新于 2020-11-25 分类于 ACM

排序

1.快速排序

1.确定分界点: q[l] q[(r + l) / 2] q[r] 随机

2.调整区间: 使得第一区间的值都小于等于x,第二区间的值都大于等于x

3.递归处理左右两段

4.基本思路: 用两个指针分别指向数组两段,分别向中间移动,当左边遇到大于x的值停下,
右边遇到小于x的值停下,如果左指针在右指针的左边,两个数交换,直到两指针相遇

5.算法模板:

void quick_sort(int q[],int l,int r)
{
    //l表示最左端下标,r表示最右端下标

    //排序完的条件
    if(l >= r)
    {
        return;
    }

    //1.找分界点,这里找q[l],i表示左指针,j表示右指针;
    int x = q[l],i = l - 1,j = r + 1;

    //2.调整区间
    while(i < j)
    {
        do{
            i++;
        }while(q[i] < x);

        do{
            j--;
        }while(q[j] > x);

        if(i < j)
        {
            swap(q[i],q[j]);
        }
    }

    //递归处理左右两段
    quick_sort(q,l,j);

    quick_sort(q,j + 1,r);

}

2.归并排序

1.确定分界点: mid = (l + r) / 2

2.递归左右两段

3.归并: 左右两段合二为一

4.临时数组的值放回原数组

5.基本思想: 递归分区,再合并

6.算法模板:

void merge_sort(int q[],int l,int r)
{
    //排序完的条件
    if(l >= r) return;

    //确定分界点
    int mid = (l + r) / 2;

    //递归左右两段
    merge_sort(q,l,mid);

    merge_sort(q,mid + 1,r);

    //归并左右两段,k为临时数组下标,i为左段下标,j为右段下标
    int k = 0,i = l,j = mid + 1;

    while(i <= mid && j <= r)
    {
    if(q[i] <= q[j]) t[k++] = q[i++];

    else t[k++] = q[j++];
    }

    while(i <= mid) t[k++] = q[i++];

    while(j <= r) t[k++] = q[j++];

    //临时数组放回原数组
    for(i = l,j = 0;i <= r;i++,j++) q[i] = t[j];

}
