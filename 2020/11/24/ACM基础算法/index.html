排序

快速排序

1.确定分界点: q[l] q[(r + l) / 2] q[r] 随机

2.调整区间: 使得第一区间的值都小于等于x,第二区间的值都大于等于x

3.递归处理左右两段

4.基本思路: 用两个指针分别指向数组两段,分别向中间移动,当左边遇到大于x的值停下,
右边遇到小于x的值停下,如果左指针在右指针的左边,两个数交换,直到两指针相遇

5.算法模板:

void quick_sort(int q[],int l,int r)
{
    //l表示最左端下标,r表示最右端下标

    //排序完的条件
    if(l >= r)
    {
        return;
    }

    //1.找分界点,这里找q[l],i表示左指针,j表示右指针;
    int x = q[l],i = l - 1,j = r + 1;

    //2.调整区间
    while(i < j)
    {
        do{
            i++;
        }while(q[i] < x);

        do{
            j--;
        }while(q[j] > x);

        if(i < j)
        {
            swap(q[i],q[j]);
        }
    }

    //递归处理左右两段
    quick_sort(q,l,j);

    quick_sort(q,j + 1,r);

}

归并排序

1.确定分界点: mid = (l + r) / 2

2.递归左右两段

3.归并: 左右两段合二为一

4.临时数组的值放回原数组

5.基本思想: 递归分区,再合并

6.算法模板:

void merge_sort(int q[],int l,int r)
{
    //排序完的条件
    if(l >= r) return;

    //确定分界点
    int mid = (l + r) / 2;

    //递归左右两段
    merge_sort(q,l,mid);

    merge_sort(q,mid + 1,r);

    //归并左右两段,k为临时数组下标,i为左段下标,j为右段下标
    int k = 0,i = l,j = mid + 1;

    while(i <= mid && j <= r)
    {
    if(q[i] <= q[j]) t[k++] = q[i++];

    else t[k++] = q[j++];
    }

    while(i <= mid) t[k++] = q[i++];

    while(j <= r) t[k++] = q[j++];

    //临时数组放回原数组
    for(i = l,j = 0;i <= r;i++,j++) q[i] = t[j];

}

二分搜索

整数二分

1.二分运用: 数据分为两个部分,求分界点

2.二分模板有两个,如果check(mid)函数是判断右段关系的,则使用模板1,
如果check(mid)函数是判断左段关系,则使用模板2 (check用来判断中间值与左右两段之一的关系)

3.基本思路: 假设目标值在闭区间[l,r]中,每次将区间长度缩小一半,当l == r时,就找到了目标值

4.算法模板1:

int bsearch_1(int l, int r)
{
    while (l < r)
    {
    int mid = (l + r) / 2;

    if (check(mid)) r = mid;

    else l = mid + 1;
    }
    return l;
}

5.算法模板2:

int bsearch_2(int l, int r)
{
    while (l < r)
    {
    int mid = (l + r + 1) / 2;

    if (check(mid)) l = mid;

    else r = mid - 1;
    }
    return l;
}

浮点数二分

1.基本思路: 同整数二分思路,在闭区间[l,r]中,每次将区间缩小一半,当r - l很小时,就找到目标
(r - l < 1e-n ,n表示题目所求精度加2)

2.算法模板:

double b_search(double l,double r)
{
    while(r - l > 1e-n)
    {
    double mid = (l + r) / 2;

    if(check(mid)) r = mid;

    else l = mid;
    }

    return l;
}

高精度

使用: 当题目所给数据特别大时,大于longlong的19位,或者数字不好操作,而字符串更好操作时,则需要使用高精度

高精度加法

1.基本思路: 使用数组来存储数据,数组第一位存储个位,以此类推,再两个数组的数据相加,大于10则进位

2.算法模板:

vector<int> add(vector<int> &q,vector<int> &p)
{
    if(q.size() < p.size()) return add(p,q);

    int t = 0;

    vector<int> c;

    for(int i = 0;i < q.size();i++)
    {
    t += q[i];

    if(i <= p.size() - 1) t += p[i];

    c.push_back(t % 10);

    t /= 10;
    }

    if(t == 1) c.push_back(1);

    return c;

}
